# FinNeo AI: Financial Strategy, Simplified.

FinNeo AI is a high-performance, **Neo-Brutalist** financial advisor platform designed to make complex investment decisions simple. Built for the modern saver, it combines AI-driven guidance with transparent pension forecasting.

## 🎨 Design Philosophy
- **Neo-Brutalist Aesthetic**: High contrast, bold black borders, hard offset shadows, and vibrant Blue/Green accents.
- **Premium Micro-interactions**: Powered by Framer Motion for a tactile, responsive feel.
- **Data Transparency**: Real-time graphing using Recharts.

## 🚀 Key Features
- **AI Advisor Bot**: Personalized investment strategy based on life stage and goals.
- **Pension Predictor**: Interactive compound interest engine for long-term retirement planning.
- **Simple Investment Catalog**: Jargon-free education on ETFs, Bonds, and Savings.

## 🛠 Tech Stack
- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **Styling**: Vanilla CSS (CSS Modules approach)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 📦 Getting Started

1. **Install Dependencies**
   ```bash
  npm install 
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
